audio tag

The <audio> element is used to embed audio content in a webpage. It supports various audio formats and provides controls for playback. 

        list of attributes in audio tag.
        controls: Adds playback controls.
        autoplay: start playback automatically.
        loop : repeat the media indefinitely.
        preload: specifies if the media should be loaded when the page loaded.

supported audio format:- mp3,wav,ogg


syntax:-

<audio controls>
              <source src="rit.mp3" type="audio/mpeg"/>
              <source src="rit.wav" type="audio/wav"/>
         </audio controls>
