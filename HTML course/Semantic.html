semantic elements

 
        <header>: represent the introductory content for a section , article, or entire web page.
        <nav> : navigation menu links would all be placed in a <nav>tag.
        <main>: the body of the page should go in main tag. there should be only one per page.
        <article>: represent an independent article on a webpage.
        < section> : represent a way of grouping together nearby content of similar theme.
        <aside> : represent the content that's less important .its mostly use sidebars.
        <footer> : represent the base of a page and section.
