Table Tag

HTML tables allow web developers to arrange data into rows and columns . we must write everything inside the table tag.

elements in table tag

        <table> Element: Defines an HTML table, which is used to organize data into rows and columns.
        <tbody> Element: group the main content(data rows) of an HTML tbl.
        <thead> Element: group the header content of an html tbl.
        <th> Element: define a header cell within a tbl.
        <td> Element: define a data cell within a tbl.
        <tr> Element: define a rows within an HTML tbl.
