what is html?

        HTML stands for hyper text markup language.
        It is standard markup language used for creating web pages.
        HTML describes the structure of a web page

Syntax of html:-

        <!DOCTYPE html >
        <html >
        <head >
        <title> PAGE TITLE </title>
        </head>
        <body>
        <h1>this is heading tag </h1>
        <p>this is paragraph tag</p>
        </body>
        </html>

EXPLAIN SYNTAX

  • The <!DOCTYPE html > declaration defines that this document is an HTML5 document.
  • The <html > element is root element of an HTML page
  • The <head > element contain meta information about the HTML page
  • The <title> elements specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab)
  • The <body>element define the documents body and is a container for all the visible contents such as headings ,paragraphs and images ,hyperlinks , tables , lists.
