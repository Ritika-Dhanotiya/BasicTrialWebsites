HTML picture tag

The <picture> HTML elements contains zero or more <source> element to offer alternative versions of an imagefor different display/device scenarious.

syntax:-

        <picture>
              <source srcset="/thapa.webp">
              <source srcset="/thapa.png">
              <source srcset="/thapa.jpg" alt=""/ >
        </picture>

HTML FIGURE TAG:-

figure tag ma image ka sath kuch caption likh sakte hai apan.

The <figure> HTML element represents self-contained content potentially with an option caption , which is specified using the <figcaption> element.

syntax:-

        <figure>
             < img src="/ritika.png" alt=""/ >
             <figcaption > Best HTML Course by ritika dhanotiya </figcaption>
        </figure>
