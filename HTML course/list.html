list tag in html

2 type of list :- 1. Ordered list and 2. Unordered list.

Ordered list <ol>

An Ordered list is used to create a list of items in a specific order , typically indicated by numbers.

syntax:-

        <ol>
             <li>First item</li>
             <li>second item</li>
             <li>third item</li>
             <li>Fourth item</li>
        </ol>

Output:-

  1. First item
  2. second item
  3. third item
  4. fourth item

Ordered list style:

        Decimal (list-style-type:Decimal;) default style:- decimal numbers.
        Lowercase Letters (list-style-type:lower-alpha;)
        Uppercase letters (list-style-type:upper-alpha;)
        lowercase roman numerals(list-style-type: lower-roman;)
        uppercase roman numerals(list-style-type: upper-roman;)

Unordered list: <ul>

An Unordered list is used to create a list of items that are not in particular order ,Each list items are marked with a bullet point.

syntax:-

        <ul>
             <li>First item</li>
             <li>second item</li>
             <li>third item</li>
             <li>Fourth item</li>
        </ul>

Output:-

  • First item
  • second item
  • third item
  • fourth item

Unordered list styles. <ul> styles:

    Disc (list-style-type: disc;): Default-style- filled circle.
    Circle (list-style-type: circle;): Hollow circle.
    Square (list-style-type: square;): Squares.
    None(list-style-type:None;): No bullet points.
